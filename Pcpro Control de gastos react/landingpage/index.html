Soluciones Técnicas Implementadas

Decisiones de desarrollo y arquitectura del proyecto

Desafío: Gestión de Estado Complejo

Manejar múltiples estados interdependientes (presupuesto, gastos, filtros, edición) de forma eficiente.

Solución: useReducer + Context API

Implementación de useReducer con Context API para un estado global predecible y acciones tipadas con TypeScript.

Desafío: Experiencia Móvil Intuitiva

Crear una interfaz táctil fluida para gestionar gastos en dispositivos móviles.

Solución: React Swipeable List

Integración de gestos de deslizamiento nativos para editar/eliminar con feedback visual inmediato.

Desafío: Validación de Presupuesto

Prevenir gastos que excedan el presupuesto disponible con validación en tiempo real.

Solución: Validación con useMemo

Cálculos optimizados con useMemo para validación instantánea y prevención de re-renders innecesarios.

Desafío: Persistencia de Datos

Mantener los datos del usuario sin backend, garantizando persistencia entre sesiones.

Solución: LocalStorage + useEffect

Sincronización automática con localStorage usando useEffect para persistencia transparente.
